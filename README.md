# 大判焼外交部ジェネレーター ver.2.4
（通常モード / ASCIIアートモード 搭載）軽量版

大判焼／今川焼／回転焼（御座候？はぁ？）の骨肉の争いなどに関するパロディ画像、  
および **ASCIIアート（AA）** を背景上に合成した画像を生成できる Streamlit アプリです。
ver2.3と比較し処理容量を軽量させました。

> ⚠ **本アプリは完全に娯楽目的のパロディ生成ツールです。**  
> 実在の団体・民族・人物とは一切関係ありません。

---

## 主な機能

### ■ 1. 通常モード（ポスター調画像生成）
- 背景画像の選択
- 本文テキスト入力
- 強調語（黄色ハイライト）の指定
- フッター（署名・日付）
- 複数フォントの選択
- テキストの自動レイアウト最適化

### ■ 2. ASCIIアートモード（AA合成）
- MS PGothic 互換フォントで AA を自然に描画
- JSON ベースの安全な文字列処理で **文字化けゼロ**
- 左寄せ表示（←添え）
- AA の見栄えを優先したフォントサイズ調整
- 通常モードのハイライト機能は自動無効化

---

## 共通機能

- HTML5 Canvas による画像生成(ver2.4で軽量化)
- JPEG 保存（画質 90%）
- X投稿リンクボタン
- NG ワードフィルタ（`.streamlit/ng_words.txt`）
- カスタムフォントの Base64 埋め込み
- `session_state` による入力内容保持
- スマホ向け最大幅 1300px 最適化

---

## 起動方法

```bash
pip install streamlit
streamlit run app.py
```
